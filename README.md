이 앱은 사용자의 30초 가량의 노래 부르는 음성을 받아서, 
해당 음성을 분석하여 기존의 DB에 존재하는 대중 음악의 음성 분석파일과 비교하여 유사도를 계산해, 사용자의 음성과 비슷한 음색의 곡을 추천합니다. 
초기 단계라 UI는 아직 없으며, 음성파일도 직접 넣어주어야 합니다.

process.py는 song.txt에 있는 음악 리스트(곡명|유튜브 영상 주소로 구성)를 읽어 해당 링크의 영상을 wav로 다운받아 임시로 temp_download에 저장합니다.
wav 파일에서 보컬 부분만 분리한 뒤, librosa와 numpy를 이용해 분석합니다.
분석은 MFCC 모델을 사용해 음성 데이터를 곡당 13개의 핵심 특징값을 추출하고, 코사인 유사도를 통해 사용자의 음성과 비교합니다.
분석한 데이터 파일은 vocal_features 폴더에 저장되어 db로써 기능합니다.
compare.py는 사용자의 음성 파일을 분석하고 유사도 기반으로 기존에 있던 곡들과 비교하여 결과를 출력합니다.
